Warcraft Classic Logs Bot

A Discord bot for Classic Warcraft Logs

Show player rankings and boss damage leaderboards, with more commands and server integrations coming soon.

Install

  2. Run ?wcl config to setup your Server and Region.

Features

Player Parses

Displays the desired player's rankings and parse average for the latest raid tier.

?wcl player Name

Boss Damage

Displays the Top 5 players of a particular class with the highest average boss damage for the latest raid tier.

?wcl bossdmg Class
